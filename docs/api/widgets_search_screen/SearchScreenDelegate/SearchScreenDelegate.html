The searchFieldHintStyle and searchFieldDecorationTheme arguments cannot both be supplied, since it would potentially result in the hint style being overriden with the hint style defined in the input decoration theme. To supply a decoration with a color, use searchFieldDecorationTheme: InputDecorationTheme(hintStyle: searchFieldHintStyle).
